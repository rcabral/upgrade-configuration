# Upgrade Configuration

Um controle de versão para o banco de dados, para que o desenvolvedor tenha o poder de migrá-lo com facilidade e confiança. Onde as evoluções de banco de dados são enviadas junto com o aplicativo e executadas automaticamente na inicialização. O que facilita o processo de entrega contínua da aplicação e favorece a ampliação da cultura de desenvolvimento ágil e devops.
